# ms-shared-api-faktura

API for the faktura service

# API

## `GET /batches`

List all batches

## `GET /batches/download`

Creates a new batch
Downloads a latin1 encoded csv-file of the given batch

## `GET /batches/:batchId/download`

Downloads a latin1 encoded csv-file of the given batch

## `GET /docs`

This readme

## `GET /new`

Returns number of new files
